Psion netBook BootLoader.
Version 011 UK

The bootloader is the program stored in netBook's Personality Module (2MB Flash chip).
It loads the OS into RAM from the CF card or via RS232 (YMODEM protocol).
Actually, unlike 5mx PRO, this is a "micro"-EPOC distributive with
a minimal set of the files + some testing utilities (can they be called?).
Internal EPOC version - 0.01(0).
Placed in LH28F160S3T-L10A (1M x 16 Flash) at 0x00000000.
Date code: 08/08/00.
